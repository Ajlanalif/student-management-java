# Student Management System (Java Evolution Project)

This repository demonstrates the evolution of a Student Management System from a Core Java console application to a Spring Boot REST API.

The goal of this project is to show understanding of:

- Core Java fundamentals
- Object-Oriented Programming (OOP)
- Layered architecture
- Exception handling
- Backend application development using Spring Boot

---

## 📂 Project Structure
student-management/
├── console-version/ # Core Java console application
├── springboot-version/ # REST API using Spring Boot
└── README.md

---

## 🔹 1️⃣ Console Version

Location: `console-version/`

A menu-driven console application built using Core Java.

### Concepts Demonstrated:
- Encapsulation
- Method overriding (`toString()`)
- Exception handling
- Java Collections (`ArrayList`)
- Lambda expressions
- Basic layered structure (Model → Service → Controller)

Data is stored in memory using an `ArrayList`.

---

## 🔹 2️⃣ Spring Boot Version

Location: `springboot-version/`

A RESTful API version of the same project built using Spring Boot.

### Concepts Demonstrated:
- REST API development
- Controller → Service → Repository layered architecture
- Spring Data JPA
- ORM mapping using `@Entity`
- Dependency Injection
- Exception handling for APIs
- Database integration (H2/MySQL)

---

## 🎯 Purpose of This Repository

This repository is designed to demonstrate the transition from Core Java application development to enterprise-level backend development using Spring Boot.

It shows understanding of how business logic can evolve from:

Console-based interaction → REST-based service architecture.
